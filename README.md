# ls

Recreating the unix command ls.
Handles the following flags:

   -> -a : shows hidden files
   
   -> -l : long listing format
   
   -> -r : reverses list
   
   -> -R : lists recursively
   
   -> -t : sorts lists according to time of last modification

CONCEPTS COVERED:

   ->Basic parsing.
   
   ->Data structures.
   
   ->Linked Lists
   
   ->Memory management
